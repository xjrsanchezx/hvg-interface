REQUIREMENTS
============
CMake 2.8.11
Qt 5.0
HVG (http://hvg-historyofvideogames.blogspot.com.es/)

BUILD
=====
Run CMake
Put into HVG_PATH variable the path to HVG. If you plan to put the exe in the same folder, just write down "./"
Generate the project and build. Tested with VS2010
